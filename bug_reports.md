## BUG-001: Нескінченне завантаження списку відділень Нової Пошти

**Статус:** Open  
**Пріоритет:** High  
**Серйозність:** Major  

### Опис
При виборі міста "Київ" у блоці доставки, список відділень Нової Пошти не завантажується (відображається лише лоадер).

### Кроки відтворення
1. Додати будь-яку книгу до кошика.
2. Перейти на сторінку Checkout.
3. Вибрати метод доставки "Нова Пошта (відділення)".
4. Ввести та вибрати місто "Київ".
5. Натиснути на випадаючий список "Оберіть відділення".

### Фактичний результат
Список порожній, постійна анімація завантаження. В консолі помилка `500 Internal Server Error`.

### Очікуваний результат
Список відділень завантажується протягом 1-2 секунд.

## BUG-002: Скидання промокоду при зміні методу оплати

**Статус:** New  
**Пріоритет:** Medium  
**Серйозність:** Minor  

### Опис
Після успішного застосування промокоду, знижка зникає, якщо користувач змінює метод оплати з "Карткою" на "Оплата при отриманні".

### Кроки відтворення
1. Додати товар у кошик.
2. Ввести валідний промокод та натиснути "Застосувати".
3. Переконатися, що ціна зменшилась.
4. Змінити спосіб оплати в блоці Checkout.

### Фактичний результат
Ціна повертається до початкової, промокод потрібно вводити повторно.

### Очікуваний результат
Знижка за промокодом має зберігатися незалежно від обраного методу оплати.
